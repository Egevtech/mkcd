# MKCD-FISH
В файле mkcd.fish описана функция mkcd для fish.
## Использование
```
mkcd <directory>
```
создает директорию, и cdхается в нее

## Установка
### Ручками
Достаточно переместить mkcd.fish в папку `/etc/fish/functions/` или `~/.config/fish/functions`

### AUR
Доступен пакет на AUR: `mkcd-fish`, например:
```shell
yay -S mkcd-fish
```
для yay, или
```shell
paru -S mkcd-fish
```
для paru
